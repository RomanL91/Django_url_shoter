--> УРЛ-шотер на Django -->

Краткое описание:
Сервис по сокрашению ссылок.
Пользователь имеет личный кабинет, в котором представлен список всех ссылок, которые
пользователь сокрашал ранее. Отдавая сервису ссылку - будет выдана в ответ короткая ссылка,
которой можно делить с другими. При клике на короткую ссылку (которую сократили) - будет
произведено обращение к сервису, где от сопоставит вызываемую ссылку с ее длинным
оригиналом с дальнейшем редиректом на оригинал.
